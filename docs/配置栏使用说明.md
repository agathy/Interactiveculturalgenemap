# 文化基因图谱 - 配置栏使用说明

## 概述

配置栏位于界面左侧，点击左上角的 **「图谱视觉参数调节」** 按钮可展开/收起面板。所有配置更改会**自动保存**到浏览器本地存储，刷新页面后依然生效。

---

## 顶部操作按钮

| 按钮 | 功能 | 说明 |
|------|------|------|
| 💾 保存 | 保存为默认 | 将当前配置保存为默认配置，下次打开页面时自动应用 |
| 🔄 重置 | 重置全部配置 | 恢复所有配置到初始默认值 |
| ⬇️ 导出 | 导出配置 | 将当前配置导出为 JSON 文件，可备份或在其他设备导入 |
| ⬆️ 导入 | 导入配置 | 从 JSON 文件导入配置，覆盖当前设置 |

---

## 一、核心标题自定义

控制根节点（中心地图）的文字标题样式。

### 1. 标题色（渐变终点）
- **功能**：设置根节点标题的颜色
- **预设**：青蓝 `#00EAFF`、金色 `#FFD700`、橙红 `#FF4500`、绿色 `#7FFF00`、白色 `#FFFFFF`
- **自定义**：点击调色盘选择任意颜色

### 2. 显示文本标题
- **开关**：控制是否显示根节点的文字标题
- **默认**：关闭（仅显示地图形状）

### 3. 字体大小
- **范围**：20px - 60px
- **默认**：32px
- **调节**：拖动滑块实时调整标题字号

### 4. 全息发光强度
- **范围**：0 - 40
- **默认**：3
- **效果**：数值越大，标题的发光效果越明显

### 5. 标题投影色
- **功能**：设置标题阴影的颜色
- **默认**：`#94e3fe`（淡青色）
- **效果**：与发光强度配合，营造全息投影效果

---

## 二、节点尺度

控制各层级节点的大小和布局参数。

### 1. 中心节点
- **范围**：100px - 600px
- **默认**：280px
- **说明**：根节点地图的显示大小

### 2. 一级节点（基因层）
- **范围**：40px - 250px
- **默认**：120px
- **说明**：五大文化基因分类节点的大小

### 3. 二级节点（目录层）
- **范围**：20px - 150px
- **默认**：75px
- **说明**：各分类下的二级目录节点大小

### 4. 三级节点（内容层）
- **范围**：10px - 100px
- **默认**：45px
- **说明**：最细粒度的内容节点大小

### 5. 节点描边开关
- **功能**：分别为中心/一级/二级/三级节点开启/关闭描边效果
- **默认**：中心节点关闭，一级节点开启，二级节点关闭，三级节点开启

### 6. 一级轨道半径
- **范围**：150px - 500px
- **默认**：280px
- **说明**：一级节点围绕中心节点的分布半径

### 7. 时间轴轨道半径
- **范围**：80px - 400px
- **默认**：180px
- **说明**：时间轴弧线与中心的距离

---

## 三、动效参数

控制各种动画效果的强度。

### 1. 装饰环自转
- **范围**：0 - 1.0
- **默认**：0.15
- **效果**：控制一级节点外围虚线圆环的旋转速度
  - 0 = 静止
  - 1 = 极速旋转

### 2. 呼吸频率
- **范围**：0 - 0.2
- **默认**：0.03
- **效果**：控制节点的呼吸动画（缩放+投影闪烁）
  - 0 = 无呼吸效果
  - 0.2 = 快速脉动

### 3. 装饰圆盘半径
- **范围**：0 - 20px
- **默认**：6px
- **效果**：控制虚线环/光晕与节点的距离
  - 0 = 紧贴节点
  - 20 = 远离节点

---

## 四、配色方案

自定义图谱的整体配色。

### 1. 大屏背景色
- **默认**：`#050a14`（深蓝黑色）
- **说明**：整个可视化区域的背景色

### 2. 中心节点投影色
- **默认**：`#082f6d`（深蓝色）
- **说明**：根节点地图的投影/光晕颜色

### 3. 时间轴颜色
- **默认**：`#00EAFF`（青色）
- **说明**：时间轴弧线的颜色

### 4. 分类配色
- **功能**：为每个文化基因分类设置专属颜色
- **影响**：
  - 一级节点颜色
  - 二三级连线颜色
  - 节点装饰环颜色
  - 中心文字颜色

**默认配色**：
- 根祖文化：`#FF6B6B`（珊瑚红）
- 忠义文化：`#4ECDC4`（青绿）
- 山河文化：`#FFE66D`（明黄）
- 古建文化：`#95E1D3`（薄荷绿）
- 酒魂文化：`#F38181`（粉红）

## 五、符号自定义

替换节点的图标为自定义图片。

### 显示中心文字
- **开关**：控制是否在一级节点中心显示文字
- **默认**：开启

### 批量上传
- **多选文件**：同时选择多个图片文件上传
- **上传文件夹**：直接上传整个文件夹内的图片
- **自动匹配**：文件名含关键字自动匹配对应节点
  - 如："山西"、"根祖"、"酒魂"、"祭祀"等

### 单个替换
- **功能**：为每个节点单独上传图标
- **格式**：支持 SVG、PNG、JPG
- **状态**：
  - "替换" = 使用默认图标
  - "已换" = 已上传自定义图标

---

## 六、图谱节点数据

管理图谱的节点结构和内容。

### 当前统计
显示当前图谱的节点数量统计：
- X 分类 / X 二级 / X 三级

### 导出节点数据
- **功能**：将当前图谱数据导出为 JSON 文件
- **用途**：备份数据、作为模板修改

### 导入节点数据
- **功能**：从 JSON 文件导入新的图谱数据
- **格式**：需符合 GraphData 接口规范
- **注意**：导入后会完全替换当前图谱

### 恢复默认节点数据
- **功能**：恢复为初始的山西文化基因数据
- **显示条件**：仅当当前数据不是默认数据时显示

## 快捷键与操作

| 操作 | 功能 |
|------|------|
| 点击节点 | 聚焦到该节点，视图平滑移动 |
| 滚轮 | 缩放视图 |
| 拖拽空白处 | 平移视图 |
| 点击空白处 | 取消时间轴聚焦 |

---

## 注意事项

1. **自动保存**：所有配置更改会自动保存到浏览器本地存储
2. **配置独立**：视觉配置和节点数据是独立的，可分别导出/导入
3. **图片处理**：上传的图片会自动裁切为圆形并添加描边（如开启）
4. **数据验证**：导入节点数据时会进行格式验证，错误会提示
5. **浏览器兼容**：推荐使用 Chrome、Edge、Firefox 最新版本

---

## 故障排除

### 配置未生效
- 检查是否点击了「保存」按钮
- 尝试刷新页面

### 导入失败
- 检查 JSON 格式是否正确
- 确认文件编码为 UTF-8
- 查看浏览器控制台错误信息

### 图片显示异常
- 确认图片格式为 SVG/PNG/JPG
- 检查文件大小（建议不超过 2MB）
- 尝试重新上传
